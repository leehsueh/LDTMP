HOW TO USE:
Put the provided SkySphere.fbx in your scene, and add the StickToCamera script on it.
Define the camera property of the script.
And finally add the SkySphere material to this object.
Play with the values.

Check the DemoScene and the assets on how to set it up.

HOW IT WORKS:
Shader calculates a gradient color based on the provided 4 different colors and height 
levels you set.
Cheap but effective.
